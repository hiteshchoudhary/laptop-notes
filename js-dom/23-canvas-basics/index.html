FILE 23: Canvas Basics

Drawing Graphics with JavaScript — shapes, text, images, and animations

Section 1 — Razorpay-Style Dashboard Bar Chart

A merchant dashboard chart drawn entirely on Canvas using rectangles, gradients, and text.

Section 2 — Drawing Shapes (Rectangles, Circles, Paths)

Canvas only has direct rectangle methods. Everything else uses paths: beginPath, moveTo, lineTo, arc.

Section 3 — Signature / Drawing Pad

Draw with your mouse or touch. Uses mousedown/mousemove/mouseup events with getBoundingClientRect().

Section 4 — Text Rendering

fillText, strokeText, font, textAlign, measureText — used for receipts, labels, watermarks.

Section 5 — Gradients & Colors

Linear and radial gradients for professional-looking backgrounds (CRED reward cards).

Section 6 — Animation (Bouncing Ball)

requestAnimationFrame syncs with display refresh (~60fps) and pauses when tab is hidden.

Section 7 — Drawing Images

drawImage has 3 forms: position, resize, crop. Images MUST be loaded first (async).
