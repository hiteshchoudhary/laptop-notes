KEY TAKEAWAYS

        
    
          
  1. Three dimension pairs: offsetWidth (total box), clientWidth (inner usable), scrollWidth (total scrollable).
    2. 
          
  2. getBoundingClientRect() gives VIEWPORT-relative position. Add scrollY for document-relative.
    3. 
          
  3. window.innerWidth includes scrollbar. document.documentElement.clientWidth is the content area.
    4. 
          
  4. ALWAYS throttle scroll handlers using requestAnimationFrame or a timer. Unthrottled = jank.
    5. 
          
  5. Debounce for resize (once after silence). Throttle for scroll (respond while scrolling).
    6. 
          
  6. ResizeObserver watches individual elements, better than window resize for most cases.
    7. 
          
  7. { passive: true } on scroll/touch listeners improves performance when you don't call preventDefault().
    8. 
          
  8. scrollIntoView({ behavior: 'smooth' }) is the simplest way to scroll to an element.
    9. 
          
  9. Combine scroll features into ONE throttled handler to minimize layout thrashing.
    10. 
          
  10. Infinite scroll = scroll position detection + async data loading + skeleton placeholders.
