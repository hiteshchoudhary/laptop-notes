Example 2 — getElementById() — The Fastest Lookup

IDs are unique, so the browser maintains a hash map making this O(1). It is the FASTEST way to select.

Example 5 — querySelector() — CSS Selector, First Match

Example 14 — Common Selection Mistakes

      
Every mistake here has been made in real codebases.

      

        
1Forgetting null check: getElementById('x').textContent crashes if ID doesn't exist

        
2Using '#' in getElementById: getElementById('#search-box') returns null

        
3Calling getElementById on an element: someDiv.getElementById('x') is TypeError

        
4Assuming querySelectorAll returns an Array: .map() is TypeError

        
5Modifying live collection while iterating forward: elements get skipped
