KEY TAKEAWAYS

        
    
          
  1. CSS transitions for simple states, CSS @keyframes for repeating, JS (rAF/WAAPI) for dynamic/complex.
    2. 
          
  2. requestAnimationFrame syncs to display refresh rate (60/120Hz), pauses in background tabs.
    3. 
          
  3. Delta time (elapsed / duration) gives TIME-BASED animation consistent across frame rates.
    4. 
          
  4. Easing functions transform linear progress into natural curves. easeInOutCubic is most natural.
    5. 
          
  5. Only animate transform and opacity -- GPU-composited, skip reflow. NEVER animate width/height/top/left.
    6. 
          
  6. element.animate() (WAAPI) returns Animation with pause(), play(), reverse(), finish(), .finished promise.
    7. 
          
  7. getAnimations() queries all running animations for bulk control.
    8. 
          
  8. Chain with await animation.finished. Parallelize with Promise.all().
    9. 
          
  9. Respect prefers-reduced-motion -- skip or reduce animations.
    10. 
          
  10. will-change: transform hints GPU layer prep. Use sparingly.
