Section 7 — Resource Hints

Start loading critical resources BEFORE the browser discovers them naturally.

Preload

This page's critical resources, load ASAP

<link rel="preload" href="/fonts/Inter.woff2" as="font">

Prefetch

Next page's resources, load during idle time

<link rel="prefetch" href="/product/123.json">

Preconnect

Establish connection to third-party domain

<link rel="preconnect" href="https://cdn.flipkart.com">

DNS Prefetch

Resolve DNS early for third-party domain

<link rel="dns-prefetch" href="https://analytics.example.com">

Section 8 — Image Lazy Loading

Images are 50-80% of page bytes. Lazy loading defers off-screen images.

Native Lazy (recommended)

<img loading="lazy" src="..." width="200" height="200">

Browser handles everything. Set width/height to prevent CLS.

IntersectionObserver (custom)

observer.observe(img) → load when 200px away

More control: custom rootMargin, placeholder swap, animation.
