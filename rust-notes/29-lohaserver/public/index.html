LohaServer

A Multi-Threaded HTTP Server Built with Rust
std only — no frameworks

How It Works

Like IRCTC handling thousands of Tatkal bookings at once, LohaServer uses a thread pool to serve multiple requests concurrently. Each incoming connection is dispatched to an available worker thread via an mpsc channel — just like passengers being directed to the next free booking counter.

Available Routes

GET / This welcome page
GET /about About this server
GET /echo Echoes request info as JSON
GET /json Sample train schedule JSON
GET /health Health check
GET /public/* Static files
Built from scratch with Rust std library.
No frameworks. No crates. Just Loha (iron).
